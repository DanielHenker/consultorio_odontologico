# Consultorio Odontológico - Dra. Edith Henker

Proyecto final del Módulo 1 del programa de desarrollo web full stack BIT.

## Descripción

Este repositorio contiene el desarrollo de un sitio web funcional para el consultorio
odontológico de la Dra. Edith Henker, especialista en rehabilitación oral con sede en
Ibagué, Tolima. Este proyecto representa el inicio de un sitio web real que será utilizado por el consultorio.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Font Awesome

## Funcionalidades implementadas

- Diseño responsivo adaptado a dispositivos móviles y de escritorio
- Catálogo de servicios con filtrado dinámico por categoría
- Vista de detalle generada dinámicamente desde parámetros de URL
- Formulario de contacto con validación de campos en JavaScript
- Navegación entre páginas con estado activo en el navbar
